# Todo Web App
**ReactJs** based todo web app.
[I'm an inline-style link](https://patwapro.github.io/todo-web-app/)

## Prerequisites
The things you need to install.
```
NodeJs and Git(optional)
```
## Installation
Just clone or download the todo-web-app
### development
run 
```
cd todo-web-app
npm install
```
then 
```
npm start
```
